redis入门简介

发表于 2018-11-10 | 分类于 redis |

API的使用和理解

通用命令

keys [pattern]命令:遍历所有key(可实用类似于正则的模式进行匹配)

dbsize命令:算出key的总数

exists key命令:检查key是否存在(存在返回1，不存在返回0)

del key1命令:删除指定的key-value(可以同时删除多个键值对)

expire key seconds命令:设置key在seconds后过期

ttl key命令:可以查看key剩余的过期时间

persist key命令:去掉key的过期时间设置

type key命令: 返回key的类型

数据结构和内部编码


单线程

redis为单线程，一个瞬间只会执行一个命令


字符串

incr key命令:key会自增1，如果key不存在，自增厚get(key)=1

decr key命令:key会自减1，如果key不存在，自减后get(key)=-1

incrby key k命令:key自增k，如果key不存在，自增后get(key)=key

decr key k命令:key自减k，如果key不存在，自减后get(key)=-k

set key value命令:不管key是否存在，都设置key

setnx key value命令:key不存在，才设置key

set key value xx命令:key存在才设置

mset key1 val1 key 2 val2命令:批量设置kv

mget key1 key2 key3 ...命令:批量获取key



getset key newvalue命令:设置新key值并返回旧值

append key value命令:将新value追加到旧value,返回追加后key的长度

strlen key命令:返回字符串的长度(注意中文)

补充命令(不常用):

incrbyfloat key value命令:增加key对应的值value,如果key不存在则设置为当前value

getrange key start end命令:获取字符串指定下表的所有值

setrange key index value设置指定下标后插入value

特点:

hset key field value命令:设置hash key对应field的value

hget key field命令:获取hash key对应的field的value

hdel key field命令:删除hash kdy对应field的value

hexists key field命令:判断hash key是否有field

hlen key命令:获取hash key field的数量

hmset key field1 value1 field2 value2命令:批量设置hash key的一批field value

hmget key field1 field2命令:批量获取hash key的一批field对应的值

hgetall key命令:返回hash key所对应所有的field和value

hvals key命令:返回hash key所对应所有field的value

hkeys key命令:返回hash key对应所有的field
