linux多命令协作:管道及重定向

管道和重定向

名称 说明 编号 默认
STDIN 标准输入 0 键盘
STDOUT 标准输出 1 终端
STDERR 标准错误 2 终端
分类 关键字 定义 例子
::: > 将STDOUT重定向到文件(覆盖) echo linux > outfile ; ls > outfile
::: >> 将STDOUT重定向到文件(追加) echo linux >> outfile ; date >> outfile
重定向 2> 将STDERR冲定向到文件(覆盖) ls nothere 2> errout
::: 2>&1 将STDERR与STDOUT结合 ls nothere 2>&1 alloutput
::: < 重定向STDIN grep linux < /etc/passwd
管道 I(竖杠) 将一个命令的STDOUT作为另一个命令的STDIN ls -l I(竖杠) grep linux ; find / -user linux I(竖杠) grep Video
