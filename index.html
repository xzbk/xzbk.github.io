引入jquery

  1. 什么是jquery？
    • jQuery 是一个 JavaScript 库。是一个“写的更少，但做的更多”的轻量级 JavaScript 库。
    • jQuery 极大地简化了 JavaScript 编程。

  2. 下载和引用

    • 下载jQuery
    • http://jquery.com提供了最新的jQuery框架下载。通常只需下载最小的jQuery包。
    • 页面引用
    • 将jQuery框架文件导入后，就可以使用jQuery的选择器和各种函数功能了

  3. Jquery函数库包含以下特性：

    • HTML 元素选取
    • HTML 元素操作
    • CSS 操作
    • HTML 事件函数
    • JavaScript 特效和动画
    • HTML DOM 遍历和修改
    • AJAX

jQuery使用步骤：

  1. 首先web项目引入js包。jQuery-2.0.3.min.js
  2. 页面导入js，
  3. 书写自己的js脚本，在js脚本中使用jQuery的选择器和函数。
